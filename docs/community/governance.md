# SINT Protocol Governance

## Overview

SINT Protocol is governed by a Technical Steering Committee (TSC) using
the SINT Improvement Proposal (SIP) process.

## Decision Making

- **Day-to-day**: Pull requests merged by maintainers (any 2 LGTM)
- **Protocol changes**: Require SIP (see SIP-0001)
- **Major versions**: TSC vote
- **Security issues**: Core team handles responsibly, coordinated disclosure
